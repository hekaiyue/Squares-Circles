Install Python 2.7 from the python website (http://python.org/)
Simply install the Python Image Library Module (PIL.exe) on your computer (included), then run BlockHero.py

ABOUT:

Block Hero is a fast-paced 2D arcade-style shooter game. While at first it looks deceptively simple, the game escalates quickly! 
Easy to learn, hard to master. Have fun!
